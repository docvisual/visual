**❌ "Injection Failed" or executor closes immediately on launch**
> Ensure Roblox is fully loaded into a game world before clicking Inject. Run `VisualExecutor.exe` as Administrator. If the issue persists, re-add the folder to your antivirus exclusion list and retry.

> This is a false positive triggered by memory-patching behavior. Add your `C:\VisualExecutor\` folder to Windows Defender's exclusion list via **Windows Security → Virus & Threat Protection → Manage Settings → Exclusions**.

**❌ Script Hub won't load or shows a blank page**

**❌ Android APK installs but crashes on launch**
> Confirm your device runs Android 10 or higher with at least 3 GB of available RAM. Close all background apps before launching. If the crash persists, reinstall the APK after a full device restart.

**❌ "Access Denied" error on Windows 11**
> Right-click `VisualExecutor.exe` → Properties → Compatibility tab → check **"Run this program as an administrator"** → Apply. This permanently runs it with elevated privileges.

**❌ Scripts throw errors despite successful injection**
